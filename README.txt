SmartCompress v2.1 - Portable Windows Release
================================================

Developed by Kamalesh S

How to run
----------
1. Extract the complete SmartCompress_v2.1 folder from the ZIP file.
2. Keep every included file in the same folder.
3. Double-click SmartCompress.exe.

No Python or FFmpeg installation is required.

Required portable files
-----------------------
- SmartCompress.exe
- ffmpeg.exe
- ffprobe.exe
- ffplay.exe
- icon.ico
- settings.json
- history.json
- smartcompress_logo.png

Important
---------
Do not move SmartCompress.exe by itself. The three FFmpeg executable files
must remain beside it for video compression to work.

SmartScreen may display a warning because this portfolio build is not signed
with a commercial code-signing certificate. Choose "More info" and then
"Run anyway" only if you received this release from a trusted source.
